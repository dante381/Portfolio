# SECURITY.md

Security checklist for the portfolio + analytics dashboard. All items tracked against the security spec in the plan.

## Threat Model

**Public surface**: Portfolio pages (anonymous traffic).  
**Private surface**: `/dashboard`, `/api/stats` — single admin only.  
**Risks**: PII leakage of visitors, admin account takeover, injection, abuse of ingest endpoint, secret leakage.

## Secret rotation procedure (S-12)

- **AUTH_SECRET**: Generate new 256-bit value (`openssl rand -base64 32`). Update in Vercel Dashboard. Redeploy. All existing JWT sessions immediately invalidated — admin must re-login.
- **HASH_SALT**: Rotates daily by design (analytics.ts reads `HASH_SALT + date`). To permanently rotate: update env var + redeploy. Cross-day continuity is not expected (privacy by design).
